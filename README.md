# Brinvex-Util-Java

### Introduction

_Brinvex-Util-Java_ is a compact library containing various helper utilities for Java.

### Maven dependency declaration
````
<repository>
    <id>brinvex-mvn-repo-public</id>
    <url>https://github.com/brinvex/brinvex-mvn-repo-public/raw/main/</url>
    <snapshots>
        <enabled>true</enabled>
        <updatePolicy>always</updatePolicy>
    </snapshots>
</repository>
        
<dependency>
    <groupId>com.brinvex.util</groupId>
    <artifactId>brinvex-util-java</artifactId>
    <version>1.8.2/version>
</dependency>
````

### Requirements
- Java 11 or above
